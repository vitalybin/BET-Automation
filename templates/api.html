Puralox Internal API

This page is just a placeholder. The application exposes internal endpoints for ELN integration (eLabFTW) and file metadata export.

  • GET /files – list uploaded files
  • POST /eln/<file_id> – push to eLabFTW
  • GET /metadata/file/<file_id> – download metadata Excel
  • GET /api/elab/templates – list ELN templates (JSON)
